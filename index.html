One-time Firebase setup

You need a free Firebase project to sync data between you and Erin. This takes about 5 minutes and is free forever.

1
Go to console.firebase.google.com and sign in with Google. Click "Add project", name it anything (e.g. my-mortgage), disable Google Analytics, click Create.
2
Inside the project, click "Realtime Database" in the left sidebar → Create Database → choose any location → start in Test mode → Enable.
3
Click the gear icon (⚙) → Project settings → scroll down to "Your apps" → click the </> Web icon → register the app → copy the firebaseConfig object shown.
4
Open index.html in a text editor, find the section labelled REPLACE THESE WITH YOUR FIREBASE CONFIG, and paste in your values.
5
Save the file, commit it to GitHub, and your dashboard will be live at yourusername.github.io/mortgage.

Firebase free tier (Spark plan) supports up to 1GB storage and 10GB/month transfer — more than enough for a mortgage tracker forever.
